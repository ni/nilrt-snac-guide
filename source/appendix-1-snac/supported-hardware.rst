------------------
Supported Hardware
------------------

The SNAC configuration is verified to work only on x64 architecture CompactRIO, sbRIO, and PXIe.
The following devices are **supported**.

- cRIO-9030
- cRIO-9031
- cRIO-9033
- cRIO-9034
- cRIO-9035
- cRIO-9036
- cRIO-9038
- cRIO-9039
- cRIO-9040
- cRIO-9041
- cRIO-9042
- cRIO-9043
- cRIO-9045
- cRIO-9046
- cRIO-9047
- cRIO-9048
- cRIO-9049
- cRIO-9053
- cRIO-9054
- cRIO-9055
- cRIO-9056
- cRIO-9057
- cRIO-9058
- PXIe-8821
- PXIe-8822
- PXIe-8840 (Dual Core)
- PXIe-8842
- PXIe-8861
- PXIe-8862
- PXIe-8880
- PXIe-8881
- sbRIO-9603
- sbRIO-9608
- sbRIO-9609
- sbRIO-9628
- sbRIO-9629
- sbRIO-9638


The following CompactRIO ARM controllers are **NOT supported**:

- cRIO-9063
- cRIO-9064
- cRIO-9065
- cRIO-9066
- cRIO-9067
- cRIO-9068
- sbRIO-9607
- sbRIO-9627
- sbRIO-9637
- sbRIO-9651
