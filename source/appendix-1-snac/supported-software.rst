------------------
Supported Software
------------------

The following NI software products have been explicitly tested and verified to work in the SNAC configuration.
Additional software support will be added to the SNAC configuration in future releases.
If your application requires software which is not on this list, please notify your NI sales representative so that we may prioritize supporting your design.

- NI LinuxRT Base System Image 2025.3 and later
- NI LabVIEW Real-Time 2022 and later
- NI-DAQmx
- NI-DCPower
- NI-DMM
- NI-FGEN
- NI-Scope
- NI-Switch

The following NI software products are **not supported** in the SNAC configurations at this time.
Products here may still be supported in future releases of the SNAC configuration.
Check the latest release of this document for the most up-to-date list of supported software.

- NI-VISA
